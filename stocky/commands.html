Stocky Command List

$!stock
Returns price of a given stock.
$!crypto
Returns price of a given crypto converted to (optional) given currency.
$!currency
Currency conversion.
$!invite
Returns Stocky's invite link.
$!ping
Returns Stocky's message ping.
$!info
Returns some data about Stocky.
$!setprefix
Change Stocky's prefix on your server Requires Permission
$!resetprefix
Resets the custom prefix your set. Requires Permission
@Stocky
Pinging the bot replies with the current prefix.
